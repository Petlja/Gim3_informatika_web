Извршавање JavaScript кода¶

JavaScript програме које пишете можете да извршавате на разне начине, а ми ћемо их овде поменути неколико. Обратите пажњу на то да се у примерима појављују различити начини приказивања података и да није могуће сваки начин приказивања користити при сваком начину извршавања програма. Уколико желите да испробате више начина извршавања, може бити потребно да прилагодите начин приказивања података.

Извршавање JavaScript кода у страницама овог курса¶

Током читања ћете наилазити на овакве прозоре, у којима можете да мењате и извршавате дате JavaScript програме (можда сте већ користили овакву функционалност учећи да програмирате на Пајтону). Програм се извршава кликом на дугме „Покрени програм”.

Функција writeln није стандардна функција језика JavaScript и она може да се користи само ако програме покрећете на овај начин (у оквиру курса).

Извршавање JavaScript кода на другим сајтовима¶

Сајтове попут http://jsbin.com, http://jsfiddle.net или http://js.do смо већ користили да испробамо функционисање HTML i CSS кода. У одговарајући прозор може се унети и JavaScript кôд, који ће одмах бити извршен.

Ако за приказ података користите функцију document.write, излаз из вашег програма ће бити уписан у веб документ. Због тога се резултат види у десном прозору, који приказује изглед документа, тј. веб странице. Ован начин је згодан док вежбате JavaScript програмирање, али имајте на уму да он утиче на садржај и коначан изглед ваше веб стране.



Ако не желите да утичете на садржај веб стране, можете да користите функцију console.log. Резултат ће се појавити у прозору који се зове конзола (console). Овај прозор можете да отворите у прегледачу притиском на тастер F12 и избором одговарајућег таба, као на слици.

Извршавање JavaScript кода у вашим веб странама¶

Основна идеја учења програмског језика JavaScript је да програме које напишете на овом језику додате вашим веб странама и тако их учините динамичнијим. Овај начин извршавања JavaScript програма је оно због чега учите JavaScript и то је оно што треба да вам остане блиско и добро познато након завршетка овог курса, јер је најближе реалној примени.

На остале начине извршавања JavaScript програма треба гледати само као на олакшицу при учењу програмирања, јер је покретање JavaScript програма на раније описане начине нешто једноставније него кроз сопствене веб стране.

Због тога свакако препоручујемо да бар повремено покрећете JavaScript програме у оквиру својих веб страна.

За овај начин рада потребан вам је било који програм за уређивање текста (едитор текста), као што је Notepad. Имајте на уму да постоји велики број напредних едитора који су бесплатни за прузимање, а који су прилагођени писању кода (на пример, VSCode, Notepad++ или Sublime).

Погледајмо сада како можете да покренете JavaScript програм у вашој сопственој веб страници. Прво направите на свом рачунару фајл program1.js са овим садржајем:

document.write('Здраво!');
document.write('<br>');
x = 2;
y = 3;
z = x + y;
document.write(z);

Затим у истом директоријуму направите фајл proba.html у коме ће се налазити овај JavaScript кôд:

<!DOCTYPE html>
<html>
  <head>
    <script type="text/javascript"  href="program1.js">
  </head>
  <body>
  </body>
</html>

Убацивањем HTML елемента script смо веб страни придружили екстерни фајл који садржи JavaScript програм. ово је слично као што смо до сада убацивањем елемента link са атрибутом rel="stylesheet" придруживали веб страни екстерне CSS фајлове са дефинисаним стилом.

Отворите фајл proba.html у веб прегледачу, као што сте то и до сада радили. JavaScript програм ће бити извршен приликом отварања. И у овом случају можете да користите функцију document.write ако желите да пишете у веб документ, односно console.log ако желите да пишете у конзолу.

Да бисте испробали писање у конзолу, направите у истом директоријуму фајл program2.js са овим садржајем:

console.log('Здраво!');
x = 2;
y = 3;
z = x + y;
console.log(z);

а затим у фајлу proba.html преправите href="program1.js" у href="program2.js".

Извршавање JavaScript кода у командном прозору¶

Да бисте ваше JavaScript програме извршавали на овај начин, потребно је да на вашем рачунару буде инсталиран NodeJs runtime (ако није већ инсталиран, можете да га инсталирате са ове адресе: https://nodejs.org/en/download). Након инсталације отворите командни прозор у директоријуму у коме се налази фајл program2.js и откуцајте node program2.js.


За овај начин извршавања потребно је само да имате JavaScript програм написан у посебном фајлу. При оваквом извршавању програма (за сада) можете да исписујете податке само помоћу функције console.log.
